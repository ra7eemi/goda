# Getting Started

Welcome to **Goda**, your fast, feature-rich, and easy-to-use Discord API library for Go.

This guide will help you set up your first Discord bot with Goda quickly.

---

## Prerequisites

Before you begin, ensure you have:

- Go **1.20 or above** installed ([Download Go](https://golang.org/dl/))
- A Discord **Bot Token** ([Create a bot here](https://discord.com/developers/applications))

---

## Installation

Run the following command to install Goda:

```bash
go get github.com/ra7eemi/goda
```
